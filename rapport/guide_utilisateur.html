Ajouter et gérer vos prospects

Ajouter un prospect manuellement

Cliquez sur Nouveau prospect dans le menu ou le bouton +.

Renseignez au minimum le nom, le budget maximum et le type de bien recherché.

Ajoutez les villes (tapez + Entrée ou cliquez sur les suggestions) et les quartiers souhaités.

Remplissez les critères détaillés : surface, pièces, état, extérieur, stationnement…

Cliquez sur Enregistrer puis confirmez si vous voulez lancer l'analyse tout de suite.

💡 AstucePlus vous remplissez de critères, plus le matching sera précis. L'observation libre est très utile pour capturer ce que les cases ne couvrent pas.

Gérer la liste de prospects

Utilisez la barre de recherche pour filtrer par nom, ville ou type de bien.

Cliquez sur ✨ pour relancer une analyse, ou sur 🔀 pour voir les matchings existants.

Archivez les prospects inactifs avec le bouton archive — ils disparaissent de la liste principale mais restent accessibles.

2

Gérer votre portefeuille de biens

Importer des biens

Via fichier Excel : Onglet Biens → bouton Importer. Le fichier doit suivre le modèle fourni.

Via synchronisation FTP (Hektor, Apimo…) : configurez les accès dans Administration → Synchronisation automatique.

Modifier un bien

Cliquez sur un bien dans la liste pour ouvrir sa fiche détail.

Utilisez le bouton Modifier pour éditer prix, surface, description ou photos.

Le lien vers l'annonce externe peut être renseigné pour ouvrir directement votre site vitrine.

⚠️ Un bien marqué comme vendu ne sera plus proposé dans les matchings. Pensez à mettre à jour le statut régulièrement.
3

Comprendre les scores de matching

L'IA analyse chaque bien par rapport aux critères du prospect et attribue un score de 0 à 100. Voici comment interpréter ces scores :

85–100
Excellent match — Le bien correspond à quasiment tous les critères. À proposer en priorité.
70–84
Bon match — Quelques critères secondaires manquent. Mérite une proposition.
55–69
Match partiel — Des compromis importants. À proposer si le portefeuille est limité.
0–54
Faible match — Non affiché par défaut. Le seuil minimum est réglable dans Paramètres.
💡 Points forts IAPour chaque matching, l'IA génère une explication des points forts et des points de vigilance, personnalisée pour ce prospect. Elle s'affiche dans la fiche matching et dans l'email de proposition.

Relancer une analyse

Cliquez sur ✨ à côté d'un prospect pour recalculer tous ses matchings. Utile après l'ajout de nouveaux biens.

L'analyse prend généralement 5 à 30 secondes selon la taille du portefeuille.

4

Envoyer une proposition par email

Depuis la page Matchings, cliquez sur l'icône ✉️ à côté d'un bien.

Vérifiez l'aperçu de l'email — il contient les photos, le prix, la description et l'analyse IA.

Modifiez le message d'introduction si besoin, puis cliquez sur Envoyer.

📧 SMTP personnel

Configurez votre propre boîte email dans Administration → Email pour que les envois partent depuis votre adresse professionnelle.

🎨 Personnalisation

Le logo, la couleur et le nom de l'agence s'appliquent automatiquement à tous les emails depuis Administration → Agence.

5

Saisie vocale d'un prospect

La saisie vocale permet de dicter les critères d'un prospect à voix haute — l'IA remplit automatiquement le formulaire.

Sur la page Nouveau prospect, cliquez sur le bouton 🎤 Remplir par la voix.

Autorisez l'accès au microphone si le navigateur le demande.

Parlez naturellement : « M. Martin, budget 300 000 euros, cherche une maison avec jardin à Le Teil ou Montélimar, 4 pièces minimum »

Cliquez sur Arrêter puis Analyser. L'IA remplit le formulaire en quelques secondes.

Vérifiez et corrigez les champs si besoin, puis enregistrez.

⚠️ La reconnaissance vocale nécessite Chrome ou Edge sur PC. Sur mobile, utilisez Chrome (Android) ou Safari (iOS).
6

Paramètres et personnalisation

🏢 Informations agence

Nom, logo, couleur, adresse, téléphone — tout ce qui apparaît dans les emails et rapports.

📊 Paramètres matching

Score minimum affiché, nombre de biens proposés par prospect, tolérance sur le budget.

📧 Configuration email

SMTP personnel pour envoyer depuis votre propre adresse. Serveur Gmail, OVH, Outlook…

🔄 Sync automatique

Connexion FTP pour importer automatiquement vos biens depuis Hektor, Apimo ou tout autre logiciel.

Ajouter des collaborateurs

Administration → Agents → Inviter un agent. Entrez son email et un mot de passe provisoire.

L'agent aura accès aux mêmes données mais ne pourra pas modifier les paramètres agence.

7

Questions fréquentes

Pourquoi mes scores sont-ils tous similaires ?

L'IA calibre les scores selon le portefeuille disponible. Si vous avez peu de biens ou des critères très larges, les scores peuvent se regrouper. Ajoutez des critères précis pour différencier davantage.

L'email n'est pas parti, que faire ?

Vérifiez votre configuration SMTP dans Administration → Email. Assurez-vous que le mot de passe d'application (Gmail) ou le mot de passe SMTP est correct. Si vous n'avez pas configuré de SMTP, les emails sont envoyés via le serveur ImmoFlash.

Puis-je importer mes prospects depuis Excel ?

Oui. Dans la page Prospects, utilisez le bouton Importer un fichier Excel. Le fichier doit contenir les colonnes : Nom, Téléphone, Mail, Bien, Villes, Budget max, Observation.

Comment supprimer les données de démonstration ?

Les prospects marqués exemple sont des données de démo. Vous pouvez les supprimer un par un via la liste des prospects, ou contacter le support pour une suppression en masse.

Le micro ne fonctionne pas

Vérifiez que vous utilisez Chrome ou Edge. Cliquez sur l'icône 🔒 dans la barre d'adresse et autorisez le microphone pour ce site. Sur mobile, vérifiez les autorisations dans les réglages de votre téléphone.
